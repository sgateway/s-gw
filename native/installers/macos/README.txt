s-gw __VERSION__ for macOS

Requirements:
- macOS 14 or newer
- Node.js 20 or newer
- An internet connection for npm dependencies

Double-click "Install s-gw.command". The installer adds the s-gw CLI,
native app, menu helper, and local services, then opens the app.

This preview build is ad-hoc signed and is not Apple-notarized. macOS may
ask you to confirm that you want to open it.

Project: https://github.com/sgateway/s-gw

s-gw __VERSION__ for Windows

Requirements:
- Windows 10 or 11
- Node.js 20 or newer
- An internet connection for npm dependencies

Double-click "Install-s-gw.cmd". The installer adds the s-gw CLI,
browser app client, tray helper, and Windows Credential Manager helper,
then completes local setup.

This is a preview PowerShell build. It has not yet been signed or validated
on the full Windows support matrix.

Project: https://github.com/barryqy/s-gw
